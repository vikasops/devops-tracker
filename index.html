🚀 Super Simple Firebase Setup (2 minutes)

Firebase is much easier than Supabase! Just follow these 3 quick steps:

1
Create Firebase Project

1. Go to console.firebase.google.com

2. Click "Create a project"

3. Name it "devops-tracker"

4. Disable Google Analytics (not needed)

2
Enable Services

1. Authentication → Get started → Sign-in method → Email/Password → Enable

2. Firestore Database → Create database → Start in test mode → Next → Done

3
Get Config

1. Project Settings (gear icon) → General

2. Scroll down → "Your apps" → Web app icon

3. Copy the config object:

const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "..."
};

That's it! Replace the config in the code and you're done! Firebase handles everything automatically - no complex SQL scripts needed! 🎉
